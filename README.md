🕸️ Quote Web Scraper

A simple and beginner-friendly Python project that scrapes quotes and author names from a website and stores them in a CSV file.

🚀 Features

 🔍 Scrapes quotes and authors from a website
 📄 Stores data in CSV format
 ⚡ Fast and lightweight
 🧑‍💻 Easy to understand for beginners

🛠️ Technologies Used

 Python
 requests
 BeautifulSoup
 CSV

📂 Project Structure

Quote-Web-Scraper/
│── scraper.py
│── data.csv
│── README.md

📊 Output

 A file named data.csv will be created
 It will contain:

   Quote
   Author
